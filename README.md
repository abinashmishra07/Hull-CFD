# Hydrodynamic CFD Analysis — Tri-Hull vs. Single-Hull Boat

Comparative drag study of two hull configurations across foiling and fully-immersed operating phases.

## What I did

- Compared viscous and pressure drag of tri-hull and single-hull configurations using ANSYS Fluent
- Used Volume of Fluid (VOF) multiphase model to accurately capture the free-surface water–air interface
- Implemented body-of-influence refinement, inflation layers and regional mesh refinement to resolve boundary layer gradients
- Parametric study of drag breakdown across hull geometries and operating phases

## Tools

ANSYS Fluent · SolidWorks

## Key findings

- VOF modelling captured free-surface behaviour accurately across both foiling and immersed phases
- Tri-hull and single-hull configurations show distinct drag characteristics depending on operating phase
- Mesh refinement strategy resolved boundary layer gradients without excessive computational cost
